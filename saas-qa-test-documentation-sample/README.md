# SaaS QA Test Documentation Sample

This folder contains a simple QA documentation sample for a SaaS web application feature. It is designed to show a clear, structured approach to manual testing, bug reporting, and test documentation.

## Feature Under Test

User Profile Management for a SaaS web application

## Included Files

- `test-plan.md`
- `test-cases.md`
- `bug-reports.md`
- `test-summary-report.md`

## Areas Covered

- Profile page access
- Profile information updates
- Field validation
- Save and cancel behavior
- Basic error handling

## Skills Demonstrated

- SaaS application testing
- Web application testing
- Manual testing
- Functional testing
- Regression testing
- Test case design
- Bug reporting
- QA documentation

## Note

This is a portfolio sample created for demonstration purposes using a fictional SaaS profile management workflow.
