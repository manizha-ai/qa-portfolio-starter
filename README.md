# QA Portfolio

This repository showcases sample QA documentation and testing artifacts that reflect my experience as a Senior Quality Engineer working with SaaS products, web applications, and mobile testing.

## Included In This Repository

- Sample test plan documentation
- Sample manual test cases
- Sample bug reports
- Sample test summary report

## Areas Highlighted

- SaaS application testing
- Web application testing
- Mobile testing
- Manual testing
- API testing
- Regression and functional testing
- Beginner-level Playwright automation with TypeScript

## Purpose

The purpose of this repository is to demonstrate a clear, structured, and practical approach to QA documentation and testing workflows through a sample SaaS feature.
