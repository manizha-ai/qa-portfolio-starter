# QA Portfolio

This repository highlights sample QA work that reflects my experience as a Senior Quality Engineer working with SaaS products and web applications.

## Featured Projects

### SaaS QA Test Documentation Sample

Location: `saas-qa-test-documentation-sample/`

This project demonstrates a structured manual testing approach for a SaaS feature, including:

- Test plan
- Test cases
- Bug reports
- Test summary report

### UI Automation Sample

Location: `ui-automation-sample/`

This project demonstrates UI automation using Playwright with TypeScript, including:

- Login flow validation
- Inventory page validation
- Add-to-cart behavior

### API Testing Sample

Location: `api-testing-sample/`

This project demonstrates API validation using Playwright with TypeScript, including:

- `GET` request validation
- `POST` request validation
- `404` response validation

## Skills Highlighted

- Manual testing
- Test case design
- Bug reporting
- SaaS application testing
- Web application testing
- API testing
- UI automation with Playwright and TypeScript

## Purpose

The purpose of this portfolio is to present clean, easy-to-review QA samples that demonstrate documentation, automation, and API testing skills in a practical way.
